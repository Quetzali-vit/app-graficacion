Oleada Zombis

Instrucciones

¡Bienvenido a Oleada Zombis!

  • Usa las flechas o las teclas 'A', 'W', 'S' y 'D' para mover al jugador.
  • Mueve el mouse para cambiar la dirección en la que mira el personaje.
  • Presiona la tecla R para recargar el arma.
  • Haz clic para disparar y elimina a los zombis.
  • ¡Sobrevive a cada ola o serás uno de ellos!
